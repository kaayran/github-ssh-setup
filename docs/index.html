github-ssh-setup

Quick SSH key setup with automated scripts

Windows

Setup Wizard Recommended

Complete setup wizard that automates the entire process: key generation, clipboard copying, and SSH config setup.

Usage:
scripts\windows\setup-github-ssh.bat
setup-ssh-key.bat

Generates a new Ed25519 SSH key pair. Creates ~/.ssh directory if it doesn't exist, prompts for account name and email.

Usage:
REM Interactive mode
scripts\windows\setup-ssh-key.bat

REM With parameters
scripts\windows\setup-ssh-key.bat personal "user@example.com"
copy-public-key.bat

Copies the public key to clipboard for easy pasting into GitHub. Displays the key content on screen.

Usage:
scripts\windows\copy-public-key.bat personal
update-ssh-config.bat

Adds or updates SSH config entry for GitHub. Creates a backup of existing config before modifications.

Usage:
REM With default host name
scripts\windows\update-ssh-config.bat personal

REM With custom host name
scripts\windows\update-ssh-config.bat personal github-work
test-ssh-connection.bat

Tests SSH connection to GitHub using configured hosts. Can test a specific host or all configured hosts.

Usage:
REM Test specific host
scripts\windows\test-ssh-connection.bat github-personal

REM Test all configured hosts
scripts\windows\test-ssh-connection.bat

Linux / macOS

Setup Wizard Recommended

Complete setup wizard that automates the entire process: key generation, clipboard copying, and SSH config setup.

Usage:
# Make scripts executable (first time only)
chmod +x scripts/linux-mac/*.sh

# Run setup wizard
./scripts/linux-mac/setup-github-ssh.sh
setup-ssh-key.sh

Generates a new Ed25519 SSH key pair. Creates ~/.ssh directory with proper permissions (700), sets 600 permissions for private key.

Usage:
# Interactive mode
./scripts/linux-mac/setup-ssh-key.sh

# With parameters
./scripts/linux-mac/setup-ssh-key.sh personal "user@example.com"
copy-public-key.sh

Copies the public key to clipboard. On Linux uses xclip or xsel, on macOS - pbcopy. Displays the key content on screen.

Usage:
./scripts/linux-mac/copy-public-key.sh personal
update-ssh-config.sh

Adds or updates SSH config entry for GitHub. Creates a backup and sets 600 permissions for config file.

Usage:
# With default host name
./scripts/linux-mac/update-ssh-config.sh personal

# With custom host name
./scripts/linux-mac/update-ssh-config.sh personal github-work
test-ssh-connection.sh

Tests SSH connection to GitHub using configured hosts. Can test a specific host or all configured hosts automatically.

Usage:
# Test specific host
./scripts/linux-mac/test-ssh-connection.sh github-personal

# Test all configured hosts
./scripts/linux-mac/test-ssh-connection.sh
Important

All scripts use only built-in system commands and require no additional dependencies (Python, Node.js, etc.). They work immediately after downloading the repository!
